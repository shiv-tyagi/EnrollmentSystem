To run the program, please follow the following steps:-
1. MAKE SURE you have a MySQL server running on your machine.
2. Please modify the settings in the `db_connect.txt` file in the root directory of the project. In most cases, you would just need to modify the password only.
3. Open terminal/command prompt on your machine and navigate to the directory where this project is present.
4. Run `java -jar EnrollmentSystem.jar`

Happy coding!!
